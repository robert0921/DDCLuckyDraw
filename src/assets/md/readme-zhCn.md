# 操作说明

下面的步骤以最简单的语言说明如何使用抽奖程序，适合不会写代码或不熟悉电脑技术的人员。

## 一次完整的抽奖流程（推荐顺序）

1) 准备数据（只需做一次）

- 打开「人员配置」页面，下载人员模板（一个表格文件）。
- 按模板把要参加抽奖的人员信息填写好（姓名、工号/编号即可）。
- 在同一页面把填好的表格上传，系统会自动导入人员。无需别的设置。

2) 设置奖项（每次活动需要确认）

- 打开「奖品配置」页面。
- 点击“新增奖项”，输入奖品名称和要抽取的人数（例如：一等奖 1 人）。
- 可以为奖品上传一张图片（用于揭晓时展示），也可以不上传。
- 设置完成后保存，右侧会显示当前奖项的已用人数（系统会在你确认后自动累加）。

3) 开始抽奖（现场操作最简单）

- 打开「抽奖页面」。页面会显示一组卡牌（格子）和一个显眼的“进入抽奖”按钮。
- 点击“进入抽奖”按钮，页面会播放一个简短的洗牌动画，随后卡牌进入可点击抽奖状态。
- 在抽奖状态下，点击任意一个未揭晓的卡牌开始抽奖。抽奖会自动运行并在几秒钟后停止并显示中奖的奖品。

4) 确认与返回

- 抽出中奖后，画面会高亮显示中奖奖品。确认无误后，点击该高亮画面即可返回到卡牌展示页（页面会自动更新已揭晓的格子）。
- 已揭晓的结果会保存下来（即便刷新页面也不会丢失）。

5) 简单恢复与重置（仅在需要重跑时使用）

- 如果你想从头开始重置所有已揭晓结果：打开浏览器地址栏，输入并运行下面两行（此操作会清空所有已揭晓记录并刷新页面）：

```js
localStorage.removeItem('luckydraw:revealedMap')
location.reload()
```

（只在你确定要清空全部记录时才执行此操作；否则不要运行）

## 小贴士

- 如果奖品图片不能显示，不会影响抽奖结果，系统会自动显示一个默认的占位图。
- 抽奖过程中如果没有声音或动画，请确认电脑的音量没有被静音，或尝试刷新页面重试。
- 建议在正式活动前先做一次完整的测试抽奖，确保人员名单和奖项都正确无误。
